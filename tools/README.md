# Tools

Local-only third-party tools used by the recovery workflow.

- `adb-fastboot/`: Android platform-tools location. This folder is incomplete until `platform-tools\adb.exe` is present.
- `downloads/`: local verified tool archives. Archives are ignored by Git.
- `ofp-extractor/`: local OFP dataset extraction launcher and notes.
- `sn-write-tool/`: SN Write Tool location for later original IMEI/NVRAM repair. This folder is incomplete until `SN_Writer.exe` is present.
- `sp-flash-tool/`: SP Flash Tool v5 location for scatter flashing. Runtime binaries are ignored by Git.

Tool binaries are ignored by Git. Commit notes, version requirements, checksums, and setup instructions.
